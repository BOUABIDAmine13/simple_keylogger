# simple keylogger
this is programme is keylgger save the infos into file text.txt.

## compile
`g++ -std=c++17 -Wall -Wextra -Werror *.cpp -o prog` 
prog is your programme name.

## execute
`prog text1.txt text2.txt`

## stop the prosess
`taskkill /F / IM prog.exe`
